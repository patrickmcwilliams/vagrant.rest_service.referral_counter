
# vagrant.rest_service.referral_counter

Creates a CentOS 7.2 server w/
  * java 1.8
  * maven 3.3


How to use:
* make sure you have vagrant installed
* open a terminal and
	* cd into the directory you cloned this repo
	* > vagrant up
	* > vagrant ssh
		* this will build the project and start the services
